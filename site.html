Começando meu site.

Nesse site você vai aprender o básico de como fazer uma página web. Primeiro precisamos baixar o Visual Studdio code, para podermos fazer alterações em HTML.
Você pode baixar ele na página official do Visual studio Code.

Agora vamos aprender o básico para criar nossa página.

Precisamos saber alguns códigos para criar nosso site, todos os códigos precisam de abas de abertura, para isso utilizamos as setas, exemplo:
< Coloque_o_código_aqui >

Alguns códigos.

< !Doctype html > para iniciar o nosso código
< html lang="pt-br" > para indicar que o documento será html e em portugues do Brasil.
< head > para criar o cabeçalho da nossa página
No head podemos inserir algumas informações como:
< meta charset="UTF-8" > para indicar que o titulo de nossa página terá todos os caracteres do pt-br.
< title > para colocarmos o título na na aba da nossa página web.
< body > para começar o corpo do nosso site, aqui todo o conteúdo irá aparecer para os usuários de nossa página.
